About

I am a software developer from Richmond VA, and a member of SEAL Team Physical Training. TrackMyTries is a personal side project I created for recording progress on the PRT and other fitness activities.

The application is written in JavaScript using AngularJS and Angular Material. The backend is Firebase, the awesome backend-as-a-service from Google. The project's code is available on GitHub.
